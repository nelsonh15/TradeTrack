# Stock Trading Journal Application

## Introduction

The Stock Trading Journal Application is a cutting-edge tool designed for traders looking to enhance their trading strategy through meticulous record-keeping and analytics. This application provides a comprehensive dashboard with advanced charting capabilities, showcasing multiple analytics to highlight ideal entries and exits for trades. Built with React components and Recharts for a dynamic and responsive frontend, and leveraging a MongoDB database for backend storage, it offers users insights into their stock trades with detailed statistics including cost basis, total value, and total percent return. Additionally, it features a stock chart with several technical indicators such as RSI (Relative Strength Index), EMA (Exponential Moving Average), and MACD (Moving Average Convergence Divergence) to aid traders in making informed decisions.

## Features

- **Dashboard**: A user-friendly interface that displays a summary of a trader's portfolio performance, including analytics on ideal trade entries and exits.
- **Charting Capabilities**: Utilizes Recharts for dynamic and interactive charting, including technical indicators like RSI, EMA, and MACD.
- **Trade Statistics**: Showcases individual trade data such as cost basis, total value, and percent return to track performance over time.
- **MongoDB Backend**: A robust database solution that securely stores user trade data and statistics for real-time access and analysis.
- **Responsive Design**: Built with React, ensuring a seamless and responsive user experience across various devices and screen sizes.

## Getting Started

### Prerequisites

Before setting up the project, ensure you have the following installed:

- Node.js (version 12.x or later)
- MongoDB (version 4.x or later)
- Yarn or npm (as your package manager)

### Installation

1. **Clone the repository and navigate to both directories (client & server) on two different terminals**

```
git clone https://github.com/nelsonh15/TradeTrack.git
cd client
cd server
```

2. **Install dependencies on both client and server directories**
```
npm install
```

3. **Start the application**
```
npm run start
```
